## でんこ用パンタ3D素材セット
駅メモ！ - ステーションメモリーズ！-に登場するキャラクター、『でんこ』達のファンアートを描くために作成した3Dモデルです。
CLIP STUDIO PAINTシリーズ向けに、CLIP STUDIO 3Dオブジェクト形式（拡張子：cs3o）データとして作成しました。


## 何に使うのか？
いずれのデータも、いくつかのでんこに共通する特徴的な形状を盛り込んでモデリングされています。
これ単体で3DCGとして使えるクオリティは担保されていません。
『でんこ』のファンアートを描く際のアタリ取り用として使用してください。
（3Dモデルを下絵やガイド代わりに、上からなぞって作画してください）

また制作ツールの関係上、曲線・曲面部品の使用箇所は限定的です。
そういった部分については適宜フリーハンドで作画いただけますようお願いします。


## 使用方法
1.ダウンロードしたzipファイルを解凍し、3D素材のファイル（拡張子：cs3o）を取り出してください。
2.作画中のCLIP STUDIO PAINTキャンバスに対し、使用したい3D素材をドラッグ＆ドロップしてください。
  通常のレイヤー上にドロップした場合は、新たな3Dレイヤーとして素材が取り込まれます。
  編集中の3Dレイヤー上にドロップした場合は、同じ3Dレイヤーに対する追加オブジェクトとして素材が取り込まれます。
3.後は一般的な3D素材と同様、任意に位置や大きさ、光源等を調整し、あなたの作画にお役立てください。
